# 畅所欲言论坛
## FreeTalk
## 项目简介
本项目是一个类似知乎的论坛应用，提供了发布话题、评论话题、点赞收藏关注私聊等论坛常用功能，另有管理界面，代码经过系统测试和压力测试。
本项目分为管理员网页前端、用户网页前端、用户移动端、后端多个部分。其中，管理员网页前端、用户网页前端使用基于React的Antd开发，用户移动端使用React Native开发，后端使用Springboot，数据库使用MySQL。
## 我在项目中的主要贡献
- 主导项目的需求分析和技术选型。
- 完成移动端前端的部分关键组件，包括话题呈现页面、导航栏、评论页面、话题浏览页面等。
- 完成后端的推荐系统。推荐系统采用基于用户的协同过滤，定时自动重新学习并生成用户画像及其推荐列表，配合自实现的持久化布隆过滤器防止重复推荐。
- 完成整个应用在Linux服务器上的部署，其中前端和后端部署在一个服务器上，数据库部署在另一个服务器上。
- 通过权限控制等保证安全性。
- 使用junit对后端进行了白盒测试，方法覆盖率达100%，语句覆盖率近100%。
- 使用jmeter对后端部分接口进行了压力测试。
- 调整所有前端的样式，使之美观大方。
